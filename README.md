# Getter

Getter is a tool written with Vue and Rust (Tauri) for create HTTP requests.

## Features

- GET

## RoadMap

- POST
- PUT
- DELETE
- PATCH
- UI Improvements

## How to install?

Clone the repo, run

```npm install```
to download modules for Vue.

Make sure you have rust and some other libraries bundle-essentials etc. already downloaded.

Then

```npm run tauri dev```

or

```cargo tauri dev```
